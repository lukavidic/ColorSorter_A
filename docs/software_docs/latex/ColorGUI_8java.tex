\doxysubsection{Detailed Description}
Controller class for the Java\+FX desktop application used to interact with microcontroller. 

This class contains the core logic of the Java\+FX-\/based desktop application. It defines and manages all UI components described in the FXML layout and handles user interactions, input validation, and communication with the microcontroller over a Wi\+Fi connection. The class includes event handlers, updates to the UI based on system state, and the sending/receiving of messages to/from the embedded system. It acts as the main bridge between the graphical interface and the backend system logic.

The application entry point and basic setup are handled separately in the \doxylink{Main_8java}{Main.\+java} file.
