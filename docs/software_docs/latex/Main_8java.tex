\doxysubsection{Detailed Description}
Entry point of the Java\+FX desktop application for controlling microcontroller. 

This class serves as the main entry point for the Java\+FX application. It loads the FXML layout file, initializes the primary window (stage) and sets up the graphical user interface. The {\ttfamily start} method uses the FXMLLoader to parse the {\ttfamily color\+\_\+gui.\+fxml} file and obtain an instance of the {\ttfamily \doxylink{classgui_1_1ColorGUI}{Color\+GUI}} controller, which handles all UI logic.

The static reference to the controller ({\ttfamily Gui\+Instance}) allows other parts of the application to access the UI logic if needed.
