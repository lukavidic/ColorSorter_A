uint8_t i2c_read1ByteRegister ( i2c_address_t address,
uint8_t reg )

Reads a single byte from a specific register of a slave I2C device.

Sends the register address in a write transaction and then performs a repeated start to read one byte. Retries the operation up to 2 times if an error occurs.

Parameters
addressThe I2C address of the slave device.
regThe register within the slave device to read from.
Returns
The byte value read from the specified register.

uint8_t reg )

Reads a 2-byte value from a specific register of a slave I2C device.

Performs a write of the register address followed by a repeated start and read of 2 bytes. The result is interpreted as little endian and returned in swapped byte order.

Parameters
addressThe I2C address of the slave device.
regThe register address to read from.
Returns
uint16_t The 2-byte value read from the register.

Reads a block of data starting from a specific register of a slave I2C device.

Performs a write of the register address followed by a repeated start and read of len bytes. The data is stored in the provided buffer. This function is non-blocking internally but blocks until completion.

Parameters
addressThe I2C address of the slave device.
regThe starting register address to read from.
dataPointer to a buffer where the read data will be stored.
lenNumber of bytes to read.
