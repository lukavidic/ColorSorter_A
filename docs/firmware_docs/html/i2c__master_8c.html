◆ i2c_close()

i2c_error_t i2c_close ( void )

Closes the I2C bus if it is not currently busy.

If the bus is not actively in use and the current address matches, this function resets the inUse flag, disables interrupts, and marks the address as free.

Returns
i2c_error_t Returns the last error status (i2c_status.error) if successfully closed, otherwise returns I2C_BUSY if the bus is still active.

◆ i2c_ISR()

void i2c_ISR ( void )

Main I2C interrupt service routine.

Clears the I2C interrupt flag, checks for address NACK conditions, and invokes the appropriate state handler from the FSM table based on the current state.

Note: Write collision flags are ignored; premature SSPBUF writes are not handled here.

◆ i2c_masterOperation()

i2c_error_t i2c_masterOperation ( bool read)

Initiates an I2C master operation (read or write).

This function begins either a master read or write transaction, depending on the read parameter. It sets the appropriate internal state, starts the I2C transfer, and chooses between interrupt-based or polled operation depending on IRQ availability.

Parameters
readIf true, initiates a read operation; if false, initiates a write.
Returns
i2c_error_t I2C_NOERR if operation started successfully, otherwise I2C_BUSY.

◆ i2c_open()

i2c_error_t i2c_open ( i2c_address_t address)

Opens the I2C bus for communication with a specific device.

This function attempts to take control of the I2C bus and set the device address for communication. It initializes internal state and callback handlers for common I2C events.

Note
If the bus is already in use (inUse flag is set), the function will return I2C_BUSY.
Parameters
addressThe 7-bit I2C address of the target device.
Returns
i2c_error_t I2C_NOERR if the bus was successfully opened, otherwise I2C_BUSY.

◆ i2c_poller()

void i2c_poller ( void )

Polling loop for I2C operations.

This function enters a loop while the I2C bus is marked as busy. It waits for I2C events and invokes the I2C interrupt service routine manually. Used when IRQs are disabled or unavailable.
